# Contributing

Thank you for your interest in contributing to the **Pokemon Type Predictor** project!

## Code Style
- Follow **PEP 8** guidelines.
- Use **type hints** for function signatures.
- Add **docstrings** to all functions and classes.
- Ensure code is readable and well-commented.

## Development Setup
1. Clone the repository.
2. Create and activate a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Pull Request Process
1. Create a new branch for your feature or fix.
2. Commit your changes with clear, descriptive messages.
3. Submit a Pull Request describing your changes and the problem they solve.

## Reporting Issues
Please use the GitHub Issues tab to report bugs or suggest features.
